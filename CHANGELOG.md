# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - Unreleased

### Added

- Sidebar chat panel with composition and session selectors
- Multi-voice conversation thread with streaming responses and thinking indicators
- `@VoiceName` mention autocomplete to target a specific voice
- Code context attachment — include active file path, selection, and error diagnostics with a message
- Right-click **Ask About Selection** command in the editor
- Status bar connection indicator with auto-reconnect (5 s retry)
- **Polyphon: Read Local API Token** command to auto-populate the API token
- VS Code settings: `polyphon.host`, `polyphon.port`, `polyphon.token`
